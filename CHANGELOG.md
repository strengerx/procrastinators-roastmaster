# Changelog

All notable changes to Procrastinator's Roastmaster will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/SemVer).

## [1.0.0] - 2026-03-10

### Added
- Initial release of Procrastinator's Roastmaster
- Core procrastination tracking functionality
- Time monitoring on social media and entertainment sites
- Customizable roast messages with escalation levels
- Browser notifications for procrastination alerts
- On-site overlay messages
- Extension popup with daily stats display
- Comprehensive settings page
- Privacy controls and data export
- Responsive UI with dark mode support

### Features
- Tracks time spent on YouTube, Facebook, Twitter, Instagram, TikTok, Reddit, Netflix, and Twitch
- Configurable warning thresholds (default: 30 minutes)
- Harsh roast escalation (default: 60 minutes)
- Daily stats reset
- User consent for data collection
- Local data storage only

### Technical
- Manifest V3 compatibility
- Chrome storage API integration
- Content script injection on targeted sites
- Event-driven architecture for tab monitoring</content>
