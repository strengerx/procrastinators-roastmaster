## 🎯 What is Procrastinator's Roastmaster?

Tired of wasting time on social media? Meet your new productivity enforcer! This Chrome extension tracks your procrastination and delivers **brutally honest, AI-generated roasts** to shame you back into productivity.

### 🌟 Key Features

- **🤖 10,000+ AI Roast Combinations**: Context-aware roasts that get more savage based on your procrastination patterns
- **📊 Sarcasm Scoring System**: Every roast comes with a sarcasm score (0-100) to measure how badly you're getting owned
- **🏆 Shame Leaderboard**: Compete with yourself! See your most unproductive days and favorite procrastination sites
- **🎖️ Productivity Achievements**: Unlock badges like "Master Procrastinator" and "Comeback Kid"
- **⚡ Real-Time Roasting**: Get roasted every 5+ minutes while actively procrastinating
- **🎨 Beautiful UI**: Modern design with dark mode support
- **🔒 Privacy-First**: Local storage only, export your data anytime

## 🚀 Quick Start

1. **Install**: [Download from Chrome Web Store](https://chrome.google.com/webstore) (coming soon!)
2. **Load Unpacked**: For development, load as unpacked extension in Chrome
3. **Start Procrastinating**: Visit YouTube, Facebook, etc.
4. **Get Roasted**: Watch the notifications and overlays appear!
5. **Check Your Stats**: Click the extension icon to see your shame leaderboard

## 🎮 How It Works

### Smart Tracking
- Monitors time spent on procrastination-prone sites
- Tracks across browser sessions
- Resets daily for fresh shame

### AI Roasting Engine
The extension uses advanced algorithms to generate personalized roasts:

```javascript
// Example roast generation
if (timeOnYouTube > 120) {
    return "You've watched more ads than actual content. Congratulations on funding the algorithm.";

```

### Achievement System
- **First Blood**: Receive your first roast
- **Hour Waster**: Waste 1+ hours in a day
- **Day Ruiner**: Waste 8+ hours in a day
- **Master Procrastinator**: Waste 100+ hours total
- **Comeback Kid**: Have a productive day after a bad one

## 🛠️ Development

### Prerequisites
- Chrome Browser
- Basic JavaScript knowledge

### Setup
```bash
git clone https://github.com/yourusername/procrastinators-roastmaster.git
cd procrastinators-roastmaster
```

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select the project folder
4. Start procrastinating to test!

### Project Structure
```
├── manifest.json          # Extension manifest
├── background.js          # Core tracking & roasting logic (includes AI roast engine)
├── content.js            # Site overlay injections
```
